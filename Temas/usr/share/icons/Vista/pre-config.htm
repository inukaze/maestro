Pre-configuration tool

 

--color

Variantes de bleu pour les icones ayant un écran de représenté

 

--start

Icone de start-here : icone du menu Gnome

 




--update

Icone du gestionnaire de mise à jour

 





--document-open

Icone des menus lors d'un Fichier > Ouvrir

 



--file-manager

Icone du gestionnaire de fichier : icone des fenêtres de l'explorateur (nautilus, konqueror, ...)

 




--app-graphics

Icone pour la catégorie Graphisme du menu de Gnome

 



--app-games

Icone pour la catégorie Jeux du menu de Gnome

 



--pref-system

Icone pour la catégorie Administration du menu de Gnome

 



--folders

Jeux d'icones de dossiers spéciaux

 

--terminal

Icone pour la console de terminal

 



--html

Icone pour les fichiers html
